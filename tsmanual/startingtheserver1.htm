Starting the server

 

After installing the server you can start the server by either double clicking the desktop icon (if you have chosen to have one on your desktop while installing the server) or double clicking the teamspeak.exe in the folder you have chosen to install the TeamSpeak server to (usually c:\Program Files\TeamSpeak2 ). Usually you will get the TeamSpeak icon in your Windows tray, which you can right click to show a couple of options.
